Frequently Asked Questions

1.What are the difference between var, let, and const?
var: Function-scoped, hoisted, can be reassigned and redeclared.
let: Block-scoped, hoisted but not initialized, can be reassigned but not redeclared.
const: Block-scoped, hoisted but not initialized, cannot be reassigned or redeclared.
2.What are the difference between `map()`, `forEach()`, and `filter()`?
map(): Transforms each element and returns a new array.
forEach(): Loops through elements and performs an action, but doesn’t return anything.
filter(): Filters elements based on a condition and returns a new array.
3.Explain arrow functions and how they are different from regular functions?
Arrow Functions: Shorter syntax, inherit this from the surrounding context, and don’t have their own arguments object. Regular Functions: Have their own this and arguments object.
4. How JavaScript Promises work?
JavaScript Promise represents the eventual completion or failure of an asynchronous operation. It has three states: pending, resolved, or rejected. Use then() to handle success and catch() for errors.
5.How closures work in JavaScript?
A closure is a function that retains access to its outer function's variables, even after the outer function has finished executing.
